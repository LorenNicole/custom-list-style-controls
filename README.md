# Custom List Style Controls

Custom List Style Controls is a WordPress Gutenberg block editor extension plugin that adds custom bullet and list-style controls to the native WordPress List block.

The plugin extends the existing `core/list` block instead of replacing it, preserving the native Gutenberg editing experience while adding additional styling options.

## Features

- Adds a List Bullet Style panel to the native List block
- Supports standard CSS list-style-type options
- Supports custom image bullets from the WordPress Media Library
- Includes bullet image size control
- Applies matching editor and frontend styles
- Built with @wordpress/scripts, React JSX, Inspector Controls, MediaUpload, SCSS, and WordPress block editor filters
